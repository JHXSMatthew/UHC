# UHC 插件优化计划

## 项目概述
- **项目名称**: UHC (极限生存)
- **类型**: Minecraft Spigot 插件
- **玩法**: 类似 PUBG 的生存竞技游戏模式
- **版本**: 1.1 Release
- **目标平台**: Minecraft 1.8.x

## 当前功能
- 团队匹配系统
- 世界边境机制
- 定时缩圈
- 观战系统
- 统计数据存储
- 支持 MySQL 数据库
